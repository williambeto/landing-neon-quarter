# Neon Quarter

Landing page para o bar fictício de arcade e pinball **Neon Quarter**.

## Sobre

Uma landing page completa, responsiva e visualmente premium com identidade de arcade noturno:
- Ambiente escuro, neon, retro-futurista
- 11 secções: Hero, Promoções, Features, Explorar, Menu, Eventos, Social Proof, Galeria, Visita, Footer
- Modal de reserva com validação
- Galeria com lightbox
- Tabs de menu e filtros de eventos
- Menu mobile funcional
- Acessibilidade: skip link, navegação por teclado, focus trap, prefers-reduced-motion
- SEO: dados estruturados, Open Graph, Twitter Card

## Tech Stack

- HTML5 semântico
- CSS3 custom properties, grid, flexbox, animações
- JavaScript vanilla (modais, tabs, filtros, scroll animations)
- Zero dependências externas
